# Progetto LSO: Tris
## Istruzioni per l'esecuzione
### Client
1\. Avviare lo script di compilazione:
```bash
sh build.sh client
```
<br>

2\. Eseguire il programma compilato:
```bash
./run_client
```
<br>

### Server (Docker)
1\. Accedi alla directory del server
```bash
cd server
```
<br>

2\. Avvia il container con Docker
```bash
docker compose up
```
